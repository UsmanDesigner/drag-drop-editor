# **Drag and Drop Playground**  

A simple jQuery-based drag-and-drop playground where users can add, move, and edit elements.

## **Features**
✔️ Drag text, images, and shapes into a playground  
✔️ Reposition elements freely within the playground  
✔️ Edit properties such as:
   - Font size, bold, italic, underline (for text)  
   - Width & height (for all elements)  
   - Change image source (for images)  

## **Technologies Used**
- **Frontend**: HTML, CSS, JavaScript  
- **Libraries**: jQuery, jQuery UI (for drag-and-drop and resizing)  

## **Setup Instructions**
1. **Clone the repository**  
   ```sh
   git clone https://github.com/UsmanDesigner/drag-drop-editor.git
   cd drag-drop-editor
   ```
2. **Open `index.html` in a browser** to start using the playground.
